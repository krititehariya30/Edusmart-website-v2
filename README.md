# EduSmart

**EduSmart** is an AI-powered learning and task management platform that helps students, teachers, parents, and admins manage tasks, track progress, and improve productivity efficiently.

## Features

- **User Authentication**
- **Task Management**
- **Performance Tracking**
- **AI Tutor Support**
- **Quiz System**
- **Notes Management**
- **Notifications**
- **Role-Based Access** (Student, Teacher, Parent, Admin)

## Tools & Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
- **ReactJS**
- **Node.js**
- **Express.js**
- **MongoDB**
- **Bootstrap / Tailwind CSS**
- **Git & GitHub**

## Purpose of the Project

**The main goal of EduSmart is to simplify learning management and improve productivity through smart task tracking and educational tools.**
